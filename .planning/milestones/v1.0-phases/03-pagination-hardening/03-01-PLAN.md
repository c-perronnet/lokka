     - If `pageResponse.status === 429`:
       - Check `retryCount >= DEFENDER_MAX_RETRIES` -- if so, throw with message including retry count and suggestion to reduce result set with $filter or $top
       - Read `Retry-After` header: `pageResponse.headers.get('Retry-After')`
       - Calculate delay: if header present, `parseInt(header, 10) * 1000` (CRITICAL: header is in seconds, multiply by 1000 for ms); otherwise `Math.min(DEFENDER_BASE_DELAY_MS * Math.pow(2, retryCount), DEFENDER_MAX_DELAY_MS)`
       - Log: `logger.info(\`Rate limited (429). Retry ${retryCount + 1}/${DEFENDER_MAX_RETRIES} after ${delayMs}ms\`)`
       - `await new Promise(resolve => setTimeout(resolve, delayMs))`
       - Increment `retryCount++`
       - `continue` (re-enter loop to retry same currentUrl)
   - After a successful response (after the existing `if (!pageResponse.ok)` block), add `retryCount = 0;` to reset the counter between pages

2. **Replace generic error with formatted error in fetchAll** (lines 322-327):
   - Replace the existing `throw new Error(\`Defender API error (${pageResponse.status})...\`)` with:
     ```typescript

     ```

3. **EU hostname rewriting on nextLink** (after line 336):
   - After `currentUrl = pageData['@odata.nextLink'] || null;`, add hostname rewriting:
     ```typescript
     if (currentUrl) {
       try {
         const nextUrl = new URL(currentUrl);
         const euHost = new URL(DEFENDER_EU_BASE_URL).hostname;
         if (nextUrl.hostname !== euHost) {
           logger.info(`Rewriting nextLink hostname: ${nextUrl.hostname} -> ${euHost}`);
           nextUrl.hostname = euHost;
           currentUrl = nextUrl.toString();
         }
       } catch (e) {
         logger.info(`Warning: Could not parse nextLink URL, following as-is: ${currentUrl}`);
       }
     }
     ```
